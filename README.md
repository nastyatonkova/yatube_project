# YATUBE_PROJECT
Social Net for bloggers

## Tech stack
- Django 2.2.19
- Python 3.9.7

## Requirements
- install virtual space
- install technical requirements
``` 
pip install -r requirements.txt
```
- proceed the command in folder with manage.py
``` 
python3 manage.py runserver
```
